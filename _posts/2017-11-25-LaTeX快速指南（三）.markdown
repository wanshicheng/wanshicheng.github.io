突然发现 LaTeX 居然自带了一份中文快速入门文档，你可以在CMD中输入：

```bash
texdoc lshort-zh
```

就可以打开《一份不太简短的 L ATEX 2ε 介绍》的文档，据说只要102分钟就可以入门。
